# 设计方案：Favicon 视觉更新 (2025-05-22)

## 1. 背景
作为项目视觉重构的第一步，需要更新 `web/public/favicon.svg`。原图标过于复杂，在浏览器标签页的小尺寸显示下效果不佳。

## 2. 目标
- 提升图标在小尺寸下的清晰度。
- 统一项目视觉语言（云盘主题）。
- 优化资源体积。

## 3. 技术规范
- **格式**: SVG (Scalable Vector Graphics)
- **尺寸**: ViewBox 0 0 32 32
- **配色**: 蓝青渐变 (#4facfe -> #00f2fe)
- **代码结构**:
  - `defs`: 定义线性渐变 `cloudGrad`。
  - `path`: 主云朵路径。
  - `circle`: 装饰性圆点。

## 4. 交付产物
- 修改后的 `web/public/favicon.svg`。

## 5. 验收标准
- [x] 图标在 32x32 比例下轮廓清晰。
- [x] 渐变效果正常显示。
- [x] 文件体积显著减小。

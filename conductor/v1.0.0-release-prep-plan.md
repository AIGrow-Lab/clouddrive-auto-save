# V1.0.0 发版准备计划

## 目标 (Objective)

在准备发布 V1.0.0 版本之前，对最近完成的一系列核心业务增强和文档更新进行汇总，并将它们写入 `CHANGELOG.md`，同时更新发版日期为今日。这确保了用户在下载最新版时能够清晰地了解自最初 Draft 发布以来的新增特性和稳定性修复。

## 具体改动 (Changes)

修改根目录的 `CHANGELOG.md`，将版本日期从 `2026-04-19` 更新为 `2026-04-29`，并追加如下主要功能：

1. **交互体验升级**：任务管理界面账号选择增加平台分组；增加分享链接快捷外跳功能，并支持自动提取、拼接并复制提取码至剪贴板，极大简化未安装网盘脚本插件时的用户操作。
2. **重命名增强**：新增 `{OLDNAME}` 魔法变量。
3. **网盘异常处理强化**：深度优化网盘接口报错逻辑。修复 139 移动云盘错误码解析导致的科学计数法异常；针对夸克网盘缺失提取码 (`41008`)、提取码错误 (`41007`, `24000`, `41009`) 等细分场景提供精确的中文 `[Fatal]` 致命错误拦截与 UI 警示，防止死循环。
4. **端到端 (E2E) 测试体系**：引入 Playwright 构建全链路测试，包含各种边缘场景的动态 HTTP Mock。
5. **构建流程增强**：增加 `e2e-test` 等管控工具于 Makefile。

## 验证方式 (Verification)

更新完成后，直接读取 `CHANGELOG.md` 确认所有的文案、格式以及发版日期正确无误。

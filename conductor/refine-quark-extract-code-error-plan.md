# 细化夸克网盘提取码错误提示计划

## 目标 (Objective)

在之前的修复基础上，进一步细化夸克网盘在解析分享链接时的提取码错误提示。目前 `41008` (未填写提取码) 和 `41007` (提取码错误) 等错误被合并处理。本计划旨在将它们拆分，提供更精确的用户反馈。

## 具体改动 (Changes)

修改 `internal/core/quark/client.go` 中的 `getStoken` 方法：

- 针对错误码 `41008`：明确提示为未填写提取码，输出 `[Fatal] 当前分享链接需要提取码，请填写提取码 (code: 41008)`。
- 针对错误码 `41007`, `24000`, `41009`：明确提示为提取码错误或无效，输出 `[Fatal] 提取码错误，请检查后再试 (code: %s)`。

由于 139 移动云盘的 `9188` 和 `200000728` 已经在之前的代码中处理完毕，这里重点聚焦于夸克网盘的精细化提示。

## 验证方式 (Verification)

1. 使用需要提取码的夸克分享链接，但不填写提取码，验证是否出现提示：“请填写提取码”。
2. 使用需要提取码的夸克分享链接，填写错误的提取码，验证是否出现提示：“提取码错误，请检查后再试”。
